# AlgoVista v1.0.0

AlgoVista is a correctness-first educational toolkit for generating, verifying and accessibly
presenting semantic algorithm traces. This candidate contains no analytics, remote service or
production dependency.

## Included

- Stable bottom-up merge sort and first-match binary search.
- Deterministic BFS and iterative DFS over bounded directed or undirected graphs.
- Independent replay verification for numeric and graph traces.
- Canonical trace import/export with strict schemas, resource limits and SHA-256 integrity.
- Responsive semantic playback for all four algorithms with keyboard controls, live status,
  reduced-motion behavior and automated accessibility regression tests.
- Four reproducible synthetic demo artifacts with source and generator lineage.
- Reproducible npm package, exact tar-member verification, isolated install smoke test and checksums.

## Verification

```bash
npm ci
npm run verify
npx playwright install chromium
npm run test:browser
```

CI runs quality gates on Node 22 and 24 and Chromium accessibility tests on Node 24.

## Integrity boundary

Artifacts and packages are not signed. SHA-256 detects altered bytes but does not authenticate the
publisher. This project visualizes bounded built-in algorithms and does not execute submitted code.
Timings and asset sizes are regression budgets, not cross-device benchmarks.
